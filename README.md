# Global COVID-19 Dashboard
An interactive web application for real-time global tracking of COVID-19 cases, recoveries, and mortality rates. Features dynamic data visualizations and regional breakdowns.
## [Global COVID-19 Dashboard ]: Data Analysis & BI Reporting
## Overview
This project leverages IBM Cognos Analytics to transform raw data into actionable business intelligence. By utilizing advanced reporting and interactive dashboards, the project identifies key trends, optimizes data exploration, and provides a clear narrative for stakeholders.

## Objectives
**Data Modeling:** Create a robust data module to ensure accurate reporting.

**Interactive Visualization:** Design intuitive dashboards for real-time data exploration.

**Business Insights:** Extract high-impact findings to support data-driven decision-making.

**Technical Mastery:** Demonstrate proficiency in IBM Cognos tools, including calculations, filters, and conditional formatting.

## Dataset Profile
**Name:**[Global COVID-19]

**Source:** [Download Dataset](./covid_19_data.csv)

**Format**: CSV

**Dimensions:** [8] columns * [306430] rows

**Domain:** Health care 

**Description:** This dataset provides a daily time-series summary of COVID-19 cases worldwide, including confirmed infections, deaths, and recoveries. It is structured to allow for longitudinal analysis of the pandemic's progression across different countries and regions, making it ideal for trend forecasting and mortality rate comparisons.

## Key Insights
**Trend 1:** Exponential Growth Phases

Confirmed cases spiked globally during specific windows (notably late 2020 and early 2022), correlating with the emergence of high-transmission variants like Delta and Omicron.

**Anomaly 2:** Recovery Rate Discrepancies

Certain regions showed "0" recoveries despite high confirmed cases. This highlights reporting inconsistencies where some health departments prioritized tracking active infections over successful recoveries.

**Prediction 3:** Seasonal Cyclicality

Based on the time-series data, infection rates show a recurring seasonal pattern. Moving forward, healthcare systems can expect predictable surges during winter months, allowing for better resource allocation.
